# 📱 Smart Motion Classification System
> 웨어러블 센서 데이터를 활용한 실시간 동작 인식 및 분류

![Motion Classification](./images/motion_classification.png)

## 📌 프로젝트 개요
스마트폰 센서 데이터(가속도계, 자이로스코프, 지자기센서)를 활용하여 사용자의 동작을 실시간으로 분류하는 IoT 프로젝트입니다.

## 🎯 주요 성과
- **계층적 분류**: 단일 모델 대비 성능 우수
- **Feature Engineering**: 전체 특성이 상위 10개보다 효과적
- **실시간 처리**: 웨어러블 환경에서 실시간 동작 인식

## 🛠 기술 스택
- **Sensors**: 가속도계, 자이로스코프, 지자기센서
- **ML**: 딥러닝, 신호처리
- **Platform**: IoT, Wearable Tech

## 📊 인식 가능 동작
- **정적 동작**: 앉기, 서기, 누워있기
- **동적 동작**: 걷기, 뛰기, 계단 오르기/내리기
- **기타**: 교통수단 이용, 휴대폰 사용 패턴
